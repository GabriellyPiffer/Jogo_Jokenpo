# 🎮 JOKENPÔ

Jokenpô é um jogo de cartas inspirado em Pedra, Papel e Tesoura com elementos extras e temática mística.

## 🃏 Cartas
- ✊ Pedra
- 📄 Papel
- ✂️ Tesoura
- 🖖🏽 Spook
- 🐛 Lagarta

## 🎯 Objetivo
Escolha cartas estrategicamente para vencer o adversário.
